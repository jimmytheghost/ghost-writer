# Ghost Writer v1.5.1 Release Notes

Release date: 2026-03-31

## Summary

`v1.5.1` is a patch release focused on markdown correctness, macOS file-open behavior, keyboard shortcut reliability, and UI polish.

## Improvements and Fixes

- Fixed markdown preview ordered-list numbering so numbered list items render with correct incremental numbers.
- Fixed macOS default `.md` file-open behavior so opening a markdown file launches Ghost Writer and loads the file in a new tab.
- Fixed Cmd/Ctrl text zoom shortcut handling for `+` and `-`.
- Added a `Check for Update` action in the About modal.
- Added an About modal button to download `llama3.1:8b`.
- Improved tab behavior:
  - tab drag reorder behavior is more reliable
  - window title reflects tab context when labels are truncated
- Fixed undo/redo behavior for mod-delete shortcuts.
- Improved copy shortcut behavior when the editor is focused.
- Improved compact model dropdown hover/border behavior.
- Hid model dropdown arrow in icon-only mode.
- Improved light-mode generation spinner contrast.
- Added a toggleable contrasting prompt hover border.
