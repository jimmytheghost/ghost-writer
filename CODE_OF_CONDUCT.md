# Code of Conduct

## Our Commitment

We want Ghost Writer to be an open, inclusive, and respectful project for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and discuss disagreements in good faith.
- Focus on ideas and code, not personal attacks.
- Give and accept feedback professionally.

## Unacceptable Behavior

- Harassment, hate speech, or discrimination.
- Personal attacks, trolling, or intimidation.
- Publishing private information without consent.
- Repeated disruptive behavior in issues, PRs, or discussions.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.
They may remove, edit, or reject comments, commits, issues, and other contributions
that violate these standards.

## Reporting

Report unacceptable behavior privately to the maintainers. Include relevant links,
screenshots, or context so the issue can be reviewed quickly.
